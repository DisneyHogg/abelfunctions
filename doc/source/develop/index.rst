Developer's Guide
=================

Abelfunctions is the primary result of `Chris Swierczewski's
<http://www.cswiercz.info>`_ Ph.D. thesis. If you are someone who is interested
in computing with Riemann surfaces, Abelain functions and algebraic curves then
you may enjoy contributing to this project. The goal of this documentation is
to get you started with adding to the software's functionality.

Development Prerequisites
-------------------------

Abelfunctions uses the `Git <http://git-scm.org>`_ version control system to
manage changes and contributions. We also use `GitHub <http://github.com>`_ to
host the `Abelfunctions project <http://github.com/cswiercz/abelfunctions>`_.
